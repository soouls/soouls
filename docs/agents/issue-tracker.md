# Issue Tracker

This repository uses GitHub Issues as the issue tracker. All work items, bug reports, and feature requests are managed directly in GitHub using the `gh` CLI.

## Usage

- To create a new issue, run:
  ```bash
  gh issue create
  ```
- To view existing issues, visit the [GitHub Issues page](https://github.com/soouls/soouls/issues).
- Ensure all issues are tagged with appropriate labels from the triage label vocabulary.
